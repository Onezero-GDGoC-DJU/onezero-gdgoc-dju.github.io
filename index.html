Build Together, Learn Together

함께 만들면서 배우는 개발 동아리

GDGoC는 프로젝트를 직접 만들고, 협업을 경험하고, 프론트엔드 기초부터 차근차근 익혀가는 실습형 동아리입니다.

참여 안내 보기 활동 살펴보기

현재 방향

  • 동아리 홈페이지를 함께 제작
  • React 기반 프론트엔드 입문
  • GitHub 이슈와 PR로 협업 연습

About

이 동아리는 무엇을 하나요?

단순히 강의를 듣는 모임이 아니라, 실제 결과물을 만들면서 프론트엔드와 협업 방식을 같이 익히는 동아리입니다.

실습 중심

아이디어를 바로 화면으로 만들고, 수정하고, 함께 개선합니다.

입문자 친화

HTML, CSS, JavaScript와 React 구조를 작은 단위부터 익힙니다.

협업 경험

이슈, 브랜치, Pull Request 흐름을 실제 프로젝트 안에서 연습합니다.

Activities

이런 활동을 함께합니다

01

기획과 구조 잡기

홈페이지에 어떤 정보가 필요한지 정하고, 페이지 구조를 함께 설계합니다.

02

컴포넌트 단위로 화면 제작

메인 화면, 소개 섹션, 카드 UI를 나누어 각자 맡아 구현합니다.

03

코드 리뷰와 개선

PR을 열고 리뷰를 주고받으며 코드와 화면을 함께 다듬습니다.

Join Us

이런 분께 잘 맞습니다

  • 프론트엔드를 처음 배워보려는 사람
  • 혼자 공부하다가 실제 협업 경험이 필요한 사람
  • 작은 프로젝트를 끝까지 만들어보고 싶은 사람

참여 방식

운영진이 만든 이슈를 선택하고 브랜치를 나누어 작업한 뒤, PR 리뷰를 거쳐 사이트에 반영합니다.
